# Browser-Based Chat Application

## Description

A real-time browser-based chat application built using Python, Flask, and Flask-SocketIO. Users can send and receive messages instantly through a web browser.

## Features

* Real-time messaging
* Multiple user support
* Username display
* Responsive chat interface
* Browser-based communication

## Technologies Used

* Python
* Flask
* Flask-SocketIO
* HTML
* CSS
* JavaScript

## Project Structure

browser_chat_app

├── app.py

├── requirements.txt

├── templates

│ └── index.html

└── static

└── style.css

## How to Run

1. Install dependencies:
   pip install flask flask-socketio

2. Run the application:
   python app.py

3. Open:
   http://127.0.0.1:5000
