# Voice Assistant

A simple voice assistant built using Python.

## Features

- Voice Recognition
- Text to Speech
- Tell Time
- Tell Date
- Open Google
- Open YouTube
- Open LinkedIn
- Search Google
- Wikipedia Search
- Open Calculator
- Open Notepad
- Stop Command
- Error Handling

## Technologies Used

- Python
- SpeechRecognition
- PyAudio
- pyttsx3
- Wikipedia

## How to Run

1. Install required libraries
2. Run:

python assistant.py

3. Speak commands through the microphone

## Example Commands

- hello
- what is the time
- what is today's date
- open google
- open youtube
- open linkedin
- search python programming
- who is virat kohli
- what is machine learning
- open calculator
- open notepad
- stop
